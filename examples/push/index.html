HTTP Push notifications

The collection below is synced through messages sent from any server.

    Run this command to update this page in real-time:

        gem install pusher
    ruby examples/push/push.rb

    The changes you push are propagated to every user viewing this page.


    Edit the file to see more examples of what you can do.

    (The commands above will only work in linux/mac systems, and required ruby and rubygems to be installed)
