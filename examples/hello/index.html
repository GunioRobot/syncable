Collection of tweets:

This collection is generated by the JS commands described.

Hello world

This is the most basic collection you can build.

This page has a container, which we will populate using JavaScript:

    
      <div id="tweets"></div>

Then we create a Collection, which autodetects the HTML container.

    
      tweets = new Collection('tweets')

Awesome. We will feed that collection with JSON items. They will be rendered using a Mustache template.

    
      tweets.template = "{{user}}: {{body}}"

We are running two basic instructions to add, update and remove elements:

  • set adds or updates an element.
  • unset removes an element.
    
      tweets.set(1, {user: "pablo", body: "Hello, world!"})
      tweets.set(2, {user: "jordi", body: "How does this work?"})
      tweets.set(3, {user: "pablo", body: "just open your console and type instructions!"})
      tweets.set(4, {user: "pablo", body: "look, i can delete this tweet"})
      tweets.unset(4)
      tweets.set(5, {user: "pablo", body: "look, i can replace this tweet"})
      tweets.set(5, {user: "pablo", body: "tweet replaced"})

You could also preload all items and render them at once, like this:

    
      tweets = new Collection('tweets')
      tweets.template = "{{user}}: {{body}}"
      tweets.items = new Hash({
        1: {user: "pablo", body: "Hello, world!"},
        2: {user: "jordi", body: "How does this work?"},
        3: {user: "pablo", body: "just open your console and type instructions!"},
        5: {user: "pablo", body: "tweet replaced"}
      })
      tweets.renderAll()
